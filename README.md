# Название проекта

TestGuru

## Объекты приложения

- Пользователь - User
  - Студент (пользователь, проходящий тесты) - Student
  - Администратор (пользователь, создающий тесты и управляющий ими) - Admin
- Тест - Test
- Вопрос (этап теста) - Question
- Ответ (на вопрос теста) - Answer
- Результат (прохождения теста) - Result
